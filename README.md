# Best January 2007 ROBLOX Client Remake
this is a redo of the best 2006 roblox client remake but with more accuracies and very obviously 2007

# whats new?
i changed a few icons, such as the group/ungroup icons and the paint bucket icon

i also added the NOWHERE-TO-BE-SEEN test category in the menu bar, this was in real roblox clients from 2006 and *maybe* even in early 2007, a screenshot of it can be found [here](https://64.media.tumblr.com/8fbeb19d608462d4b70da939ebdbe928/5dab48bcf578d9c2-c9/s540x810/12d7f9faf66b00d7f70a37ed742dd81d66d629e0.png).

there is legacyroblox support built right into the AppSettings.xml file so you don't have to edit it to go to legacyroblox

there is 2 startplayer scripts now, one simulating jan 2007, and one simulating feb 2007, jan 2007 removes the toolbox and edit mode buttons, and swaps around the help and fullscreen button, while feb 2007 doesn't do anything

there is now <b>OBVIOUSLY</b> animations added to this client

there is now a setup file that simulates what it would look like to install this as if it was a real roblox version, and is also an alternate way to use the client if you want to use it like that
